## Como Enviar
- Acesse o link onde está o e-mail.
- Selecione toda a área pressionando Ctrl+A.
- Após selecionar tudo, pressione Ctrl+C para copiá-lo.
- Por último, cole no corpo do e-mail pressionando Ctrl+V.
